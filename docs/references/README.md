# Referencias visuales

Carpeta para imágenes/capturas de referencia de dirección de arte (moodboard,
patrones de fondo, hero, geometría orbital, layouts editoriales).

Vacía por ahora — los patrones de esta fase se derivaron de descripciones
textuales y del análisis de sitios de referencia. Para incorporar referencias:
soltar los archivos aquí (`.png`/`.jpg`/`.webp`) y citarlos en el PR o en
`docs/VISUAL-SYSTEM.md`. No se usan imágenes de terceros en el sitio; son solo
material interno de diseño.
